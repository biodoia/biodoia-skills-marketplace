## Palette Usage Guidelines

### Choosing a Palette

1. **Cyberpunk Neon** -- Default for most retro-modern UIs. High energy, versatile, excellent contrast.
2. **NES / Game Boy / CGA** -- Use when specifically emulating hardware aesthetics or building retro game UIs.
3. **SNES Warm** -- Good for RPG-themed applications, warm and inviting.
4. **Synthwave** -- Best for landing pages, portfolio sites, music-related projects.
5. **Amber / Green Terminal** -- Perfect for CLI tools, dashboards, monitoring UIs, hacker themes.

### Combining Palettes

You can mix palettes by using one as the base and pulling 2-3 accent colors from another:
- Cyberpunk Neon base + NES Red for a retro-danger state
- Green Terminal base + Cyberpunk Magenta for important highlights
- Synthwave base + CGA Yellow for bright accent elements

### Contrast Ratios

Key contrast ratios against `#0a0a0a` (void black):
- `#ff00ff` (magenta): **6.2:1** -- WCAG AA pass (large text AAA)
- `#00ffff` (cyan): **16.3:1** -- WCAG AAA pass
- `#39ff14` (neon green): **14.5:1** -- WCAG AAA pass
- `#ff1493` (hot pink): **5.8:1** -- WCAG AA pass (large text AAA)
- `#e0e0ff` (lavender white): **17.2:1** -- WCAG AAA pass
- `#8888aa` (muted lilac): **6.5:1** -- WCAG AA pass (large text AAA)
